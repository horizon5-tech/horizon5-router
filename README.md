# Horizon5 Router

Django-based API router for the Horizon5 algorithmic trading framework.

## Main Project

This is a companion service to the main framework:  
**[horizon5-connect](https://github.com/horizon5-tech/horizon5-connect)** - Algorithmic trading framework for backtesting and executing trading strategies

## Overview

API service providing REST endpoints for managing backtests, orders, reports, and snapshots from the Horizon5 trading framework.

## Features

- RESTful API controllers for trading data
- MongoDB integration for data storage
- Celery task queue support
- Repository pattern architecture
- Django REST framework

## Stack

- Python 3.x
- Django
- MongoDB
- Celery
- Docker
